## ジェイソン・ステイサムについて（知らない人のために）

ジェイソン・ステイサムが映画で演じるキャラクターは、たいてい何かの専門職だ。

運び屋、殺し屋、メカニック、元特殊部隊員——。ふわっとした「なんかすごい人」ではなく、必ず「何ができるか」が明確な人物だ。その専門性があるから、彼は強い。何者かわからない人間に、誰も仕事を頼まない。

今にして思えば、あれはClaudeへの指示の話でもあった。

なぜ僕の頭の中にいるのかは、今回も引き続き、わからない。

---

## 「一度に一つを深く」の次

前回の学びは、手ごたえがあった。

「全部やってください」をやめて、「この一点を深く」に変える。それだけで、Claudeから返ってくるものが変わった。提案書の差別化ポイントについて聞いたとき、読んで5分で使えるものが出てきた。

しばらくそれを続けていた。一点を深く。一点を深く。

でも、どこかまだ引っかかりがあった。

たとえばコピーを書いてもらったとき。確かに悪くない文章が返ってくる。でも「なんか、普通だな」という感覚がある。どこかの誰かに向けて書かれた、汎用的な文章。僕のクライアントに向けて書かれた感じがしない。

そんなある午後のことだった。

ステイサムがドアのそばに立っていた。スーツ姿だった。ドアを開ける前に、一度だけ横から隙間を覗き込んでいた。それから静かに開けた。何も起きなかった。いつも何も起きない。

「何かが足りない」と僕は言った。

「何が」

「コピーを頼むと、悪くないんだけど、普通なんだ」

ステイサムはこちらを向いた。「役割を与えているか」

「役割？」

「Claudeに、誰として答えるかを伝えているか」

「……伝えていない」

「そのままでは死ぬぞ」

## 「専門家として」事件

なるほど、役割を与えればいいのか。早速やってみた。

```
専門家として、このWebサイトのトップページのコピーを
改善してください。
```

Claudeが答えた。

……普通だった。

さっきと大して変わらなかった。「より多くのお客様に価値を届ける」みたいな、どこかで見たような言い回しが並んでいた。

「専門家として、と書いた」と僕はステイサムに言った。

「どんな専門家だ」

「…専門家」

ステイサムが少し目を細めた。その表情で理解した。「専門家」というのは役割ではなかった。肩書きの形をした、ただの形容詞だった。

「Claudeには想像する素材が必要だ」とステイサムは言った。「どんな経歴で、何を大切にしているプロなのか。それを渡さないと、Claudeは『なんとなく詳しそうな誰か』にしかなれない」

「つまり、役割も具体的に書けってこと？」

「試してみろ」

## 別人になった瞬間

今度はこう書いた。

```
あなたはWebコピーライティングの経験が10年あり、
中小企業のブランディングを専門としているコピーライターです。
感情に訴えるより、具体的な数字や事実で信頼を積み上げる
文体を得意としています。

この前提で、以下のWebサイトのトップページのコピーを
改善してください。
```

Claudeが答えた。

読み始めて、3行で気づいた。

**違う。**

文章の粒度が、さっきと全然違った。「より多くの」ではなく「導入から3ヶ月で」。「価値を届ける」ではなく「作業時間を週4時間削減した」。数字と事実で積み上げていく、確かに「文体」と呼べるものがあった。

「……なんか、別人みたいだ」と僕は言った。

ステイサムは何も言わなかった。

## 細かくすればするほど変わっていく

面白くなってきた。

試しにもう一つ。同じコピーを、今度は違う役割で頼んだ。

```
あなたはユーザー体験（UX）を重視するWebデザイナーで、
「読まれないWebサイト」の問題を15年解決してきた人物です。
訪問者の離脱を防ぐ観点から、このトップページのコピーを
評価してください。
```

また、違った。

今度は文章ではなく、指摘が来た。「このセクションは読まれない位置にある」「この一文は意図が伝わる前に終わっている」——デザイナー目線の、鋭い指摘だった。

同じコピーを渡した。同じClaudeが答えた。でも、答えた「人物」が違うと、全く違うものが返ってくる。

そのとき、何かがつながった気がした。

「これって……役割だけじゃないよな」と僕は言った。

ステイサムが少し振り向いた。

「第2話で学んだ『具体的に状況を伝える』も、今日の『役割を細かく渡す』も、結局同じことだ。要件を細かく伝えれば伝えるほど、Claudeの出力が細かくなっていく」

「……続けろ」

「逆に言えば、返ってきたものが漠然としているのは、Claudeのせいじゃない。渡した情報が漠然としていたから、漠然としたものが返ってきた。それだけだ」

ステイサムはしばらく黙っていた。

部屋の隅に視線を走らせた後、窓のカーテンを静かに閉めた。外から見えないようにか、それとも外を見えなくするためか。どちらかはわからなかった。

「お前はわかってきた」と彼は言った。

「また『わかった、は危険だ』って言いますよね」

「言う。だが今回は——まあいい」

## 急に、ペーパーナイフが出てきた。

少し自信がついてきた感覚があった。

「要件を細かく。役割を具体的に。これを続ければ——」

突然、ステイサムの手に何かが現れた。光を反射する薄い刃だった。

「…それは」

「ペーパーナイフだ」

「なんで」

「封筒を開ける」

「封筒なんてないですよ、ここに」

ステイサムはゆっくりとペーパーナイフをしまった。「…そうか」

「……で、何でしたっけ」

「要件の話だ」

「そうでした」

「続けろ」

## とばりが今日気づいたこと

役割指定は、「Claudeに人格を与える」というより、「答えてほしい視点と専門性を指定する」ことだった。

そして振り返ると、第2話の「具体的に状況を伝える」も、今日の「役割を細かく渡す」も、根っこは同じだった。

**細かく伝えるほど、Claudeの出力が細かくなる。**

当たり前のようで、やってみないとわからなかった。細部のデザインは、こちらが渡した情報のデザインをそのまま反映する。Claudeは鏡だ——渡したものの精度を、そのまま返してくる。

ステイサムはその日、珍しく長く残っていた。夜になっても、まだ部屋の片隅にいた。何かを考えているように見えた。何を考えているかはわからなかった。聞かなかった。

翌朝、彼はいなかった。

テーブルの上に、一本のペーパーナイフが置いてあった。

封筒は、なかった。
